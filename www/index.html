Как использовать эти файлы на itch.io

Чтобы добавить эти HTML-файлы на твою страницу itch.io, тебе нужно:

  1. Разместить эти файлы на веб-сервере (например, GitHub Pages)
  2. Добавить iframe с ссылками на эти файлы в описание твоего профиля на itch.io

Пример кода для вставки в описание профиля:

<!-- Анонс нового плагина Building System -->
<iframe src="https://твой-домен.com/building_system_announcement.html" style="width:100%; height:680px; border:0; margin:0; padding:0"></iframe>

<!-- Интерактивные кнопки -->
<iframe src="https://твой-домен.com/tosha_interactive_buttons.html" style="width:100%; height:400px; border:0; margin:0; padding:0"></iframe>

<!-- Карточка плагина -->
<iframe src="https://твой-домен.com/building_system_card.html" style="width:100%; height:500px; border:0; margin:0; padding:0"></iframe>

Замени "https://твой-домен.com/" на реальный URL, где ты разместишь эти файлы.
